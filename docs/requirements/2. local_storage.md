# REQ-002: 生词本管理

## 概述

在现有翻译浮窗基础上，增加生词本交互功能。用户可通过红心按钮收藏/取消收藏单词，通过标签切换已掌握/未掌握状态。同时提供独立的生词本管理窗口，支持浏览、搜索、编辑、删除操作。

## 与 REQ-001 的关系

- REQ-001 的"自动记录"行为不变 — 所有查过的词仍自动写入 `wordbook.json`
- 本需求在此基础上新增 `favorited`（收藏）字段，红心控制该字段
- `mastered`（已掌握）字段已存在，本需求为其增加 UI 交互
- 已掌握的词再次双击时正常弹窗翻译，显示已掌握标记

## 功能详情

### F1: 翻译浮窗 — 红心收藏按钮

- 翻译浮窗中新增红心按钮（位于单词旁或发音按钮附近）
- 首次查询的单词：红心为空心（☆），表示未收藏
- 点击空心红心 → 变为实心（★），标记为已收藏
- 再次查询已收藏的单词：红心为实心
- 点击实心红心 → 变为空心，取消收藏
- 收藏状态实时写入 `wordbook.json`

### F2: 翻译浮窗 — 已掌握标签

- 翻译浮窗中新增"已掌握"切换按钮（位于浮窗底部或单词旁）
- 未掌握状态：显示"标记为已掌握"文字按钮
- 已掌握状态：显示"✓ 已掌握"标签（绿色），可点击切换回未掌握
- 掌握状态实时写入 `wordbook.json`

### F3: 生词本管理窗口

- 通过系统托盘菜单打开（新增"生词本"菜单项）
- 独立窗口，非浮窗（有标题栏、可拖拽、可调整大小）
- 功能：
    - **单词列表：** 显示所有记录的单词，包含：单词、翻译、音标、查询次数、收藏状态、掌握状态
    - **搜索/筛选：** 按单词搜索，按状态筛选（全部 / 已收藏 / 已掌握 / 未掌握）
    - **编辑：** 可修改单词的翻译、例句、备注
    - **删除：** 可删除单条记录
    - **排序：** 按查询时间、查询次数、字母顺序排序

## 数据结构变更

`wordbook.json` 中的 `WordEntry` 新增字段：

```json
{
  "word": "hello",
  "translation": "你好",
  "phonetic": "həˈloʊ",
  "examples": [
    "Hello, how are you?"
  ],
  "query_time": "2026-02-28T10:00:00Z",
  "last_query_time": "2026-02-28T10:00:00Z",
  "source_context": "",
  "query_count": 3,
  "mastered": false,
  "favorited": false
}
```

- `favorited`: 是否收藏（布尔值，默认 false）
- `mastered`: 已存在，默认 false

## 验收标准

1. 翻译浮窗中显示红心按钮，可切换收藏/取消收藏
2. 翻译浮窗中可切换已掌握/未掌握状态
3. 收藏和掌握状态变更后立即写入 `wordbook.json`
4. 再次查询同一单词时，浮窗正确显示其收藏和掌握状态
5. 已掌握的词双击后仍正常弹窗，显示已掌握标记
6. 系统托盘可打开独立的生词本管理窗口
7. 管理窗口可浏览、搜索、筛选、编辑、删除单词
8. 管理窗口的修改实时同步到 `wordbook.json`
9. 不影响 REQ-001 已有功能（取词、翻译、发音、托盘开关等）
